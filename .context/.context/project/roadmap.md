# Somatic-Canticles Roadmap

Summary of the 8-week roadmap. Full detail: `.docs/01-Planning/roadmap/Roadmap.md`

## Vision

Webapp that uses biorhythm cycles to unlock chapters of embodied consciousness, guiding users through the WitnessOS framework via sound, breath, and somatic practice.

## Phase 1: Foundation (Weeks 1-2)

**Goal:** Project scaffolding and core systems

- **Week 1:** Initialize webapp repo (Next.js + TypeScript), Kanban, chapter architecture (12 chapters), biorhythm calculation engine spec.
- **Week 2:** Biorhythm calculator (from birth data), chapter data structure, database schema (user progress), audio playback foundation, component library skeleton.

**Deliverable:** Working biorhythm display + chapter list

## Phase 2: Content Creation (Weeks 3-4)

**Goal:** Record and produce first 6 canticles

- **Week 3:** Physical & emotional cycles — record Morning Octave, Terpsichore's Dance, Earth Resonance, Heart Harmonics; write chapter content for 6 chapters.
- **Week 4:** Intellectual & spiritual — record Architect's Code, Urania's Navigation, 44-Sequence; guided meditations; unlock animations.

**Deliverable:** 6 complete canticles + chapter content

## Phase 3: Unlock System (Weeks 5-6)

**Goal:** Biorhythm → chapter mapping + user experience

- **Week 5:** Biorhythm → chapter matching, unlock trigger system, chapter progression tracker, unlock ceremony (13-second), notifications.
- **Week 6:** Biorhythm visualization wheel, chapter map view, progress analytics, daily check-in flow, journal/reflection.

**Deliverable:** Functional unlock system + dashboard

## Phase 4: Polish & Launch (Weeks 7-8)

**Goal:** Beta testing and public release

- **Week 7:** Mobile responsive, audio player enhancements, animation polish, performance, accessibility audit.
- **Week 8:** Beta testing (13 users), feedback, marketing, documentation, public launch (44-day cycle).

**Deliverable:** Public webapp + 13 beta testers
