- **Claim:** ACTH release triggers adrenal cortex to release cortisol.
  - *Status:* Valid.
- **Claim:** Adrenal medulla handles "quick-fire thunder" (adrenaline/epinephrine).
  - *Status:* Valid.
- **Claim:** Cortisol return should signal satiation (negative feedback loop).
  - *Status:* Valid.
- **Claim:** Inversion of feedback loop (Cortisol signals "failure" -> more ACTH).
  - *Status:* Valid description of HPA axis dysregulation (e.g., in PTSD/depression).
